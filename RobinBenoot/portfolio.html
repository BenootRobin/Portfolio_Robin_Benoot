Portfolio

In het eerste jaar van het graduaat Programmeren moesten we voor het vak User Interfaces een portfolio ontwikkelen. Dit hebben we gedaan a.d.h.v. onze kennis van HTML/CSS die we tijdens de lesperiode hadden opgedaan.

Het doel van dit project was eigenlijk om te starten met de basics van HTML/CSS om een basiswebsite op te maken. Hierna moesten we dan gaandeweg ons project aanpassen naargelang wat we elke week hadden bijgeleerd om uiteindelijk op een volwaardige website te komen.

Uiteindelijk is het dan ook het doel om het portfolio dat we nu hebben ontworpen verder uit te breiden in de opvolgende vakken.
